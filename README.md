# iexec-common

## Overview

The iexec-common library holds classes related to:
* iExec tasks and replicates
* Results of iExec tasks
* Compute model
* Blockchain-related things like
    * web3 clients
    * iExec Proof-of-Contribution
* Docker client
* TEE configurations
* Utils tooling

## Build from sources

```
./gradlew build --refresh-dependencies
```
