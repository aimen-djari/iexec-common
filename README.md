# iexec-common

## Overview

The iexec-common library holds classes related to:
* iExec replicates
* Results of iExec replicates
* Compute model
* Blockchain related things
* TEE configurations
* Utils tooling

## Build from sources

```
./gradlew build --refresh-dependencies
```
